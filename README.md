## Built with

- Plain HTML & CSS — no frameworks
- Hosted for free with **GitHub Pages**
- Fonts via Google Fonts (Syne, Inter, JetBrains Mono)

---

## How to update the site

1. Edit `index.html`
2. Open **GitHub Desktop**
3. Write a short commit message describing the change
4. Click **Commit to main** → **Push origin**

The live site updates within about 60 seconds.
