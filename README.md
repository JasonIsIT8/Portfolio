# Jason's Portfolio

> Personal portfolio and resume site — built with HTML/CSS, hosted on GitHub Pages.

🌐 **Live site:** [yourusername.github.io](https://yourusername.github.io)

---

## What's inside

| Tab | Description |
|-----|-------------|
| **Overview** | Introduction, current skills, and what I'm working on |
| **Portfolio** | Project cards, work experience timeline, and certifications |
| **Resume** | Printable one-page resume (PDF-ready) |
| **Log Work** | Password-protected form to add projects, skills, experience, and certifications — updates all tabs automatically |

---

## Built with

- Plain HTML & CSS — no frameworks
- Hosted for free with **GitHub Pages**
- Fonts via Google Fonts (Syne, Inter, JetBrains Mono)

---

## How to update the site

1. Edit `index.html`
2. Open **GitHub Desktop**
3. Write a short commit message describing the change
4. Click **Commit to main** → **Push origin**

The live site updates within about 60 seconds.
